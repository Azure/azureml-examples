
How to Run this AutoML Job with CLI (Image Object Detection)
======================================================================


1. Prepare data/images (Upload images to Workspace and generate JSONL dataset)
------------------------------------------------------------------------------

You first need to prepare the data: The image based datasets need to be uploaded and later the JSONL dataset needs to be generated based on the uploaded images URIs. 

You do this by simply running the prepare_data.py available in this folder, with a command like the following:


/> python prepare_data.py --subscription [YOUR_AZURE_SUBSCRIPTION] --group [YOUR_AZURE_RESOURCE_GROUP] --workspace [YOUR_AZURE_WORKSPACE]




2. Run the AutoML CLI command pointing to the cli-automl-image-object-detection-task-fridge-items.yml file in this folder plus the Azure ML IDs needed:
----------------------------------------------------------------------------------------------------

Note: Your compute cluster should be GPU-based when training with Images or Text. 
You need to specify/change the name of the cluster in the .YAML file (compute: azureml:gpu-cluster).

/> az ml job create --file ./cli-automl-image-object-detection-task-fridge-items.yml --workspace-name [YOUR_AZURE_WORKSPACE] --resource-group [YOUR_AZURE_RESOURCE_GROUP] --subscription [YOUR_AZURE_SUBSCRIPTION]




3. If you would like to use the Automode feature, which performs an automatic hyperparameter sweep, then run the AutoML CLI command pointing to the cli-automl-image-object-detection-task-fridge-items-automode.yml file in this folder plus the Azure ML IDs needed:
----------------------------------------------------------------------------------------------------

Note: Your compute cluster should be GPU-based when training with Images or Text. 
You need to specify/change the name of the cluster in the .YAML file (compute: azureml:gpu-cluster).

/> az ml job create --file ./cli-automl-image-object-detection-task-fridge-items-automode.yml --workspace-name [YOUR_AZURE_WORKSPACE] --resource-group [YOUR_AZURE_RESOURCE_GROUP] --subscription [YOUR_AZURE_SUBSCRIPTION]
